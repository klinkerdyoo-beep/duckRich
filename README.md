#  DuckRich web บันทึกรายรับรายจ่าย

เป็นเว็บที่พัฒนาเอง เพื่อ ฝึกฝนทักษะด้าน UX/UI Design เเละ Full stack

### ที่มาเว็บ
เคยใช้บันทึกรายรับรายจ่ายในโทรศัพท์ เเล้วรู็สึกว่าอยากให้มีในคอมบ้างค่ะ ยังไม่เจอเว็บบันทึกรายรับรายจ่ายในคอมที่ถูกใจถามเพื่อน เพื่อนเล่าให้ฟังว่าจดใน Excel หน้าตาดูไม่น่าใช้เลยค่ะ เลยอยากลองทำเว็บนี้ขึ้นมา
ส่วนตัวชอบเว็บที่ดูน่ารักๆ เลยออกเเบบ UX/UI ให้น่ารัก น่าใช้ค่ะ
